## Get Your Site Running in 5 Minutes!

### Step 1: Install Dependencies
```bash
cd son2-latin-music
npm install
```

### Step 2: Start Development Server
```bash
npm run dev
```

Open http://localhost:3000 - Your site is live!

### Step 3: Add Your Media Files

1. **Photos**: Drop your band photos into `public/images/`
2. **Audio**: Add MP3 files to `public/audio/`
3. **Videos**: Place MP4 files in `public/videos/`

### Step 4: Deploy to Vercel (FREE!)

1. Create account at vercel.com
2. Install Vercel CLI:
```bash
npm i -g vercel
```

3. Deploy:
```bash
vercel
```
